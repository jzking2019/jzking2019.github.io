到今天為止，在不斷編寫及不斷調試並修復目前所發現的bug下，落寒的個人網站正式推出！

此網站有以下四大亮點：

 · 能根據瀏覽器設定自動調整主題，或者手動設定主題

 · 菜單中有五個選單，方便使用者交互操作

 · 能夠根據設備自動適配，提升使用者體驗

 · 最佳化網頁代碼，使網頁加載速度更勝一籌

預計上線功能：

 · 商店（若有合適的第三方聯盟營銷平台合作，將會推出）

 · 捐助我們（若有合適的第三方捐款，將會推出）
